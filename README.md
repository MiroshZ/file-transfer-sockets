# File Transfer Sockets

Клиент-серверное приложение на Python для передачи файлов по TCP-сокетам.

## Возможности

- LIST — список файлов на сервере
- UPLOAD — загрузка файла на сервер
- DOWNLOAD — скачивание файла с сервера
- EXIT — завершение соединения
- поддержка нескольких клиентов одновременно

## Запуск

### Сервер
```bash
python3 server.py
python3 client.py
